Set up your Github

Github is an extremely popular company that makes it very easy to host git repositories. “git” is a utility and paradigm of working on code together. Read about it here: http://git-scm.com/book

Github is a for-profit company that makes it easy to host code for anyone in the world (or privately within an organization) to collaboratively work on together. It happens to heavily use the “git” protocol for everything it does.

Almost all major open source projects are hosted on Github. So much so, that it has effectively become the epicenter of modern open-source development. Because so many software developers use Github, a person’s individual publically viewable Github profile has become a very good way to see what projects one has contributed to, what one is interested in, and how good one is at developing. The fact you can actually see clear examples of someone’s work has made Github profiles a remarkable résumé replacement. Your Github profile will likely be a very important part of your software development career going forward — even more-so than a paper résumé.

Set up your Github account

  • Go to www.github.com and make an account.

  • Remember to keep your username and email professional

  • The norm is for it to include some combination of your first and last names/initials.

Point Your Domain to Github

  • Now you need to "point" your domain to github, so it can show the world your page

  • Change the "A-Record" point to the following Ip Address: 204.232.175.78

  • Test your account by seeing if you get a 404 error from github

  • Check out this Guide from github if you are having trouble

  • Every domain provider has a different way of doing this - search your provider to find out how to change a record.

  • It may take a day for these changes to be implemented - That's OK. Keep moving forward
