Lists

Types of Lists

  1. Ordered Lists

  2. Unordered Lists

  3. Definition Lists

Ordered Lists

Also known as numbered lists, each item in the list is numbered.

  1. start with the < ol > tag
  2. end with the < /ol > tag
  3. each item is wrapped in < li > and < /li > tags

Example Ordered List

	
 <ol>
	<li>List item 1</li>
	<li>List item 2</li>
	<li>List item 3</li>
 </ol>

Unordered Lists

Also known as bulleted lists, each item in the list has a bullet.

  1. start with the < ul > tag
  2. end with the < /ul > tag
  3. each item is wrapped in < li > and < /li > tags

Example Unordered List

	
 <ul>
	<li>bullet item 1</li>
	<li>bullet item 2</li>
	<li>bullet item 3</li>
 </ul>

Nested Lists

Lists inside of lists!
You can start a second list inside your first list

  1. start in your < li > tag
  2. start a new < ul > before closing the < li >
  3. add your nested list remembering to close the initial < /ul > tag at the VERY end

Example Nested List

	
<ul>
  <li>bullet item 1</li>
  <li>bullet item 2
    <ul>
      <li>nested item 1</li>
      <li>nested item 2</li>
    </ul>	
  </li>
  <li>bullet item 3</li>
</ul>

Links

Links are why the Web is so awesome!

they are what makes surfing possible (no wetsuit required)

  1. link from one website to another
  2. link from one page in your site to another
  3. link from a specific part of a page to another
  4. open a new window with a link
  5. send an email with a link

Link Tags

Always an open and close

<a>   </a>

This is the base of how we link to anything - what we add between those two tags is what we will link to. Remember you close your links!

Add HREF

We tell the browser where to link by adding

href = "..."

Example

<a  href  = " http://www.website.com ">This is your Link <a>

Link to Other Sites

We need to use an absolute url (don't forget the http or https-copy and paste from your browser to be sure)


<a href="http://www.website.com">This is your Link<a>

Open a Link in a New Window

we add an attribute inside the a tag of target="_blank"


Generally we don't want to open links in new windows
(let the user decide if they want to do this)


 <a href="about.com" target="about.com"> About Page <a>

Link to a page in your site

We need to tell the browser what .html document to link to/open


If all the pages (.html files) are in the same folder you can just use the name of the file


<a href= "about.com">About Page<a>

Typical Folder Structure

Directory Relationships

If it's in the same folder
= "contact.html"


If it's inside another folder inside your folder
= "images/logo.png"


If it's outside the folder you are in (if we were inside the images folder and wanted to link to index.html)
= "../index.html"


Anchors

how we link to specific part of a page

How anchors work

They use the id attribute- great for 1-page scrolling sites

  • decide where on the page you want to link

  • add the id=".." attribute inside the opening tag

  • the attribute name should start with a letter or _ and be relevant to the text

  • can't reuse attribute names (ids only get used)

  • in the nav link to the anchor text using href="#id-name"

Images

Getting images into your site

The Image Tag

images are defined with the < img > tag.


<img src  = " file location "/>

images have no closing tag.
The / is located at the end of the tag

The Alt Attribute

The required alt attribute specifies an alternate text for an image, if the image cannot be displayed.

<img  src="lama.png" alt="big llama">

You decide what the description text is.
The alt attribute is required for your code to validate.
